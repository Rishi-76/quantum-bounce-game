# QUANTUM BOUNCE // 2D Physics Arcade 🕹️

A fast-paced, high-performance 2D arcade survival game built completely from scratch using vanilla JavaScript and the HTML5 Canvas API. The engine utilizes live physics vector manipulation to dynamically invert gravity on user input.

## 🧠 Core Engineering Frameworks
* **Dynamic Kinematic Math Engine:** Implements continuous gravitational acceleration vectors ($v = v_0 + at$) to handle jumping trajectory and directional orientation shifts seamlessly.
* **AABB Collision Matrices:** Real-time calculation of Axis-Aligned Bounding Box intersections at 60Hz intervals to handle microsecond object boundary collisions.
* **Structural Garbage Collection Loop:** Features an automated array tracking pipeline that dynamically clears spent off-screen hazard nodes from system memory to prevent RAM memory leaks during runtime.
* **Local High Score Persistency:** Integrates the browser's native window `localStorage` engine to store and display permanent high-score values across separate browser instances.

## 🛠️ Architecture Stack
* HTML5 (Canvas element deployment rendering)
* CSS3 (Deep dark-mode theme structure)
* JavaScript (ES6 custom entity models, input binding captures, and synchronization paint frames)
